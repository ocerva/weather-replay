# 🌦️ WeatherReplay

A modern historical weather comparison app built with Next.js, TypeScript, and Open-Meteo APIs.

Compare weather conditions between different dates and locations around the world with elegant charts, visual insights, and a premium responsive interface.

---

## ✨ Features

- 🌍 Historical weather search
- 📅 Date-to-date weather comparison
- 📊 Interactive charts
- 🌙 Dark mode
- 📱 Mobile-friendly responsive design
- 🔍 City autocomplete
- 🚀 Live deployment with Vercel
- 🔗 Share functionality

---

## 🛠️ Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Recharts
- Open-Meteo API
- Vercel

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/ocerva/weather-replay.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
